reinforcement learning for RTL synthesis

Model-written Verilog, graded by compiling and simulating it against a reference testbench.

Cologic samples Verilog from a policy model and grades each sample with the Verilator simulator. The reward is the fraction of testbench vectors whose outputs match the reference. The benchmark below reports pass@1 on the held-out tasks.

How the reward is computed
1 · sample
Policy model writes a Verilog module for the task spec + interface.
2 · compile + simulate
Verilator builds the candidate beside the golden reference and drives random vectors.
3 · score
reward = 0.10·compiles + 0.90·(outputs matched / total). pass@1 counts only full matches.
